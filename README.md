# Cold-War-

A React-based interactive and illustrated website that explores the events and causes leading to the end of the Cold War and the fall of the Soviet Union.

## Features

- **Historical Highlights**: Covers major events such as the Chernobyl disaster, proxy wars, and Mikhail Gorbachev's policies.
- **Responsive Design**: Optimized for desktop and mobile viewing.
- **Interactive Elements**: Uses animations powered by Framer Motion for a modern user experience.

## Key Technologies

- **React**: A JavaScript library for building user interfaces.
- **Framer Motion**: For animations and smooth transitions.
- **Tailwind CSS**: For a sleek, responsive design.
